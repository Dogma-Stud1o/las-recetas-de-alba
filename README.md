# Las Recetas de Alba

Sitio de presencia de la panadería familiar en **Siguatepeque, Honduras**.

Pan artesanal desde 1945 — semitas, bollitos y encargos de la semana con entrega los sábados.

## Local

```bash
npx serve .
```

Abrir la URL que imprima el servidor (raíz del proyecto).
